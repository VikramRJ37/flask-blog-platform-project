About this site

Built to write.
Built to read.

The Inkwell is a personal blogging platform built entirely with Python and Flask. Every route, form, model, and view was crafted from scratch — no shortcuts, no CMS, just clean backend logic and a love for the written word.

"The best writing platform is the one you build yourself."

This project focuses on getting the fundamentals right: user authentication, session management, image uploads, paginated post listings, and full CRUD for blog posts. The UI is intentionally lean — the content is the star.

Built with

Python Flask SQLAlchemy Flask-Login WTForms Bootstrap 5 Jinja2
